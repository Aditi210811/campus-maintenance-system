# Campus Maintenance & Issue Tracking System

A web-based application that allows students to report campus issues and administrators to manage and resolve them.

## Features
- User & Admin authentication
- Issue reporting with image upload
- Status tracking (Open / In Progress / Resolved)
- Date & time tracking
- Color-coded status
- Persistent storage

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser localStorage (prototype backend)

## Demo
Run using Live Server or open index.html in browser.
